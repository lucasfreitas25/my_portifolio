# Portfólio Lucas Freitas — Dev & Data Scientist 

Bem-vindo ao meu portfólio profissional! Este projeto foi desenvolvido para apresentar minha experiência, habilidades e projetos nas áreas de **Desenvolvimento de Software** e **Ciência de Dados**.

**Acesse online:** [lucasfreitas25.github.io/my_portifolio](https://lucasfreitas25.github.io/my_portifolio/)

## Tecnologias Utilizadas

Este projeto foi construído com tecnologias modernas para garantir alta performance, responsividade e uma excelente experiência de usuário:

- **[React](https://reactjs.org/)** com **[TypeScript](https://www.typescriptlang.org/)**
- **[Vite](https://vitejs.dev/)** para build e desenvolvimento super rápidos
- **[Material-UI (MUI)](https://mui.com/)** para componentes estilizados e responsivos
- **[i18next](https://www.i18next.com/)** para internacionalização (Suporte nativo a Português e Inglês)
- **Three.js** e **React Three Fiber** para suporte a renderização 3D
- **Estilização** customizada utilizando a engine do MUI com toques de glassmorphism

## Funcionalidades Principais

- **Internacionalização (i18n)**: Troca instantânea de idiomas entre Português e Inglês.
- **Design Moderno e Dark Mode**: Interface focada em uma estética premium e minimalista.
- **Totalmente Responsivo**: Layout que se adapta perfeitamente a dispositivos móveis, tablets e monitores widescreen.
- **Seções Dinâmicas**:
  - **Hero Section**: Animações de background e introdução rápida.
  - **Sobre Mim**: Apresentação profissional detalhada.
  - **Skills e Experiência**: Lista categorizada de tecnologias (Frontend, Backend, BD), seção dedicada para certificações (com links) e histórico profissional (Seduc, Unimed, etc.) apresentados em *Dialogs* estilizados.
  - **Projetos**: Divisão inteligente em categorias (Dev, Data Science, Design) com links para repositórios e visualização direta dos dashboards no Power BI.

## Como executar o projeto localmente

Siga os passos abaixo para rodar a aplicação em seu ambiente de desenvolvimento:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/lucasfreitas25/my_portifolio.git
   ```

2. **Navegue até a pasta do projeto:**
   ```bash
   cd my_portifolio
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. Acesse no seu navegador: `https://lucasfreitas-portifolio.netlify.app`
